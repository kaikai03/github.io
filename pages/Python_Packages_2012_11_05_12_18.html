Python Packages

记录一下常用的Python Packages

Math

  • numpy，数值计算扩展，存储和处理大型矩阵及计算，http://www.numpy.org/
  • theano，基于表示矩阵向量的numpy和包含大量数学功能的scipy,提供符号表示的数学公式。https://pypi.python.org/pypi/Theano
  • pandas，数据分析包，大量库、标准的数据模型、操作大型数据集所需的工具，https://pypi.python.org/pypi/pandas/
  • scikit-learn，机器学习模块，http://scikit-learn.org/stable/
  • ggplot，简易坐标图生成， http://ggplot.yhathq.com/

GUI

  • Matplotlib ，相当于ggplot进阶版，https://pypi.python.org/pypi/matplotlib
  • selenium，自动测试，https://pypi.python.org/pypi/selenium
  • PyQt，界面库，https://pypi.python.org/pypi/PyQt5/

Web FrameWorks

  • Tornado，http://www.tornadoweb.org/en/stable/
  • Flask，http://flask.pocoo.org/
  • Web.py，http://webpy.org/
  • Pelican，静态博客框架，https://github.com/getpelican/pelican

Tools

  • xlrd，Excel处理，http://www.python-excel.org/
  • lxml，http://lxml.de/
  • configparser，ini文件解析，https://docs.python.org/2/library/configparser.html
  • uuid，https://docs.python.org/2/library/uuid.html
  • shortuuid 生成简洁，明确，URL安全的UUID。访问 https://github.com/stochastic-technologies/shortuuid
  • msgpack-python，类似JSON的一个数据序列化，https://pypi.python.org/pypi/msgpack-python/
  • psutil，一个用于获得处理器和系统相关信息的模块，https://github.com/giampaolo/psutil
  • Boltons 一个纯 Python 工具集，超过 100 个实用工具，https://github.com/mahmoud/boltons

Database

  • MySQL-Python，https://pypi.python.org/pypi/MySQL-python
  • pymongo，https://github.com/mongodb/mongo-python-driver
  • redis，https://pypi.python.org/pypi/redis/
  • SQLAlchemy，SQL工具包及对象关系映射（ORM）工具，http://www.sqlalchemy.org/
  • peewee， SQL工具包及对象关系映射（ORM）工具，https://pypi.python.org/pypi/peewee
  • torndb，Tornado原装DB，https://github.com/bdarnell/torndb

Net

  • requests，最好用的http工具，http://www.python-requests.org/
  • grequests，Requests + Gevent，https://github.com/kennethreitz/grequests
  • gevent，高并发的网络性能库，http://www.gevent.org/
  • twisted，基于事件驱动的网络引擎框架，https://twistedmatrix.com/trac/

System

  • sh，系统管理，https://pypi.python.org/pypi/sh

SSH

  • Fabric，Python库和命令行工具，用于连接到SSH服务器并执行命令，http://www.fabfile.org/
  • paramiko，遵循SSH2协议，支持以加密和认证的方式，进行远程服务器的连接，http://www.paramiko.org/

Files

  • Pathlib，文件和路径处理 https://pathlib.readthedocs.org/en/pep428/
  • watchdog，Python库和Shell实用程序来监视文件系统事件，https://github.com/gorakhargosh/watchdog

Date

  • Arrow，好用的时间处理库，http://crsmithdev.com/arrow/
  • when.py，友好的时间日期库，https://github.com/dirn/When.py

Image

  • PIL，Python Imaging Library，处理图像，http://www.pythonware.com/products/pil/
  • Pillow，是 PIL 的替代版本，http://python-pillow.github.io/

Spider

  • PyQuery ，解析网页，https://pypi.python.org/pypi/pyquery
  • beautifulSoup，网页分析，https://pypi.python.org/pypi/beautifulsoup4
  • Scrapy，爬虫框架，http://www.scrapy.org/

Other

  • Jinja2，模板引擎，https://pypi.python.org/pypi/Jinja2
  • virtualenv，虚拟环境，https://pypi.python.org/pypi/virtualenv/
  • virtualenvwrapper，多环境拓展，https://pypi.python.org/pypi/virtualenvwrapper-powershell/
  • libmc，高效轻便的C++/Python Memcached 客户端库，https://github.com/douban/libmc
  • Celery，分布式任务调度模块，http://www.celeryproject.org/
  • supervisor，进程管理工具，http://supervisord.org/

String

  • xpinyin 汉字转拼音的库，https://github.com/lxneng/xpinyin
  • python-user-agents 提供了一个简单的方法通过解析（浏览器）用户代理字符串来确定，如移动电话，平板电脑及其能力的设备，https://github.com/selwin/python-user-agents
  • jieba 中文分词组件，https://pypi.python.org/pypi/jieba/

RESTful APIs

  • Eve 一款Python的REST API框架，用于构建和部署高可定制的、全功能的RESTful的Web服务，http://python-eve.org/
  • Django REST 框架可以轻松部署web APIs，其是一个聚健壮性与弹性于一体的web工具包，http://www.django-rest-framework.org/
  • Flask-RESTful 基于Flask框架开发的Rest APIs。访问：https://github.com/flask-restful/flask-restful
