webRtc转发服务简单安装配置

过程概要

安装过程就不赘述了，反正都是 configure、make、install；

1.数据库

支持mysql，Redis，MongoDB；

以MongoDB为例：编译安装，然后，新建mongodb.conf并写入

dbpath=/usr/local/mongodb/db
logpath=/usr/local/mongodb/logs/mongodb.log
port=27017
fork=true
nohttpinterface=true

启动

/usr/local/mongodb/bin/mongod --bind_ip localhost -f /usr/local/mongodb/bin/mongodb.conf

2.数据库驱动

编译安装 MongoDB-C-Drive；

3.coturn

编译安装coturn；

4.turnserver配置

拷贝coturn里的演示配置覆盖工程配置，内含演示用的签名证书；

sudo cp coturn/example/* /usr/local/etc/

修改配置文件：

vim /usr/local/etc/turnserver.conf

打开、修改以下选项：

listening-port=3478
listening-ip=140.112.xxx.xxx
relay-ip=140.112.xxx.xxx
relay-threads=5
lt-cred-mech
mongo-userdb="mongodb://localhost:27017/coturn" 
realm=junze.com
max-bps=10000000

其中lt-cred-mech，该加密方式为必须打开选项、因为webRtc要求的，否则不会产生reply的candidate；

5.启动turnserver

turnserver

不带任何参数启动它会去加载那一堆etc里的默认文件；

7.创建turn帐号

turnadmin -a --mongo-userdb="mongodb://localhost:27017/coturn" -u userid -r junze.com -p pwd

8.完

以上，基本就完成了，接着是在前端js里写入turn地址了。

var iceServer = {                                                            
        "iceTransportPolicy": "relay",                                       
        "iceServers": [                                                      
                {"urls":"stun:180.153.xxx.xxx"}
                {"urls":"turn:userid@180.153.xxx.xxx","credential":"pwd"}
        ]                                                                    
    };

其他

非关键的东西：

1.查看帐号密码信息： 进入mongo shell

/usr/local/mongodb/bin/mongo coturn                                                                                                      
db.turnusers_lt.find()db.turnusers_lt.find()

返回大概如下数据： 

{ "_id" : ObjectId("57441b21c89507f577071401"), "name" : "userid", "realm" : "junze.com", "hmackey" : "6e1c2b0553be00ee9f09ca46f46xxxxxxx}

2.turnsrver自带的测试工具

turnutils_stunclient -p 3478 140.112.xxx.xx                
turnutils_uclient -v -t -T -u userid -w pwpwd80.153.xxx.xxx

3.修改防火墙 

vi /etc/sysconfig/iptables

-A INPUT -p tcp -m state --state NEW -m tcp --dport 30000:65000 -j ACCEPT
#随便写的端口范围

service iptables restart
